🎯 هدف المهمة الأساسي:
تحليل جميع ملفات المشروع المرتبطة بصفحة الهبوط (Landing Page) بدقة، مع التركيز على الترابط الداخلي بين الأكواد والوظائف، للتأكد من أن أي خلل في التعديلات السابقة أو مشاكل استنساخ الدوال يتم اكتشافها قبل البدء بأي تعديل جديد.

📁 1. قراءة جميع الملفات وتحليلها ملفًا ملفًا:
افتح وراجع:

ملف/ملفات HTML

ملف/ملفات CSS

ملف/ملفات JavaScript

أي ملفات إضافية تؤثر على الصفحة (إن وجدت)

👁️‍🗨️ 2. فهم وظيفة كل ملف بدقة:
تحديد دور كل ملف:

هل يحتوي عناصر واجهة المستخدم؟

هل يتعامل مع التفاعل مع المستخدم (أزرار، خيارات، إدخالات)؟

هل يعالج البيانات أو يُجري عمليات حسابية؟

هل يُخفي أو يُظهر عناصر؟

هل يرسل بيانات (مثلاً إلى تليجرام أو إلى قاعدة بيانات)؟

🔗 3. تتبّع الترابط بين الملفات:
تحليل العلاقة بين:

عناصر HTML وملفات JavaScript

تحميل واستدعاء ملفات CSS

طريقة ظهور أو إخفاء العناصر بناءً على التفاعل

دوال JavaScript التي تعمل عند الأحداث (onclick, onchange…)

مسارات إرسال البيانات (submit أو API Calls)

🧠 4. فحص منطق عمل الصفحة بالكامل:
التأكد من:

ظهور العناصر بناءً على تفاعل المستخدم

منطق احتساب السعر الكلي (إن وُجد)

تخزين بيانات المستخدم عبر المتغيرات

التحقق من صحة المدخلات (Validation)

كيفية إرسال البيانات (Form Submit أو عبر fetch/XHR)

ما إذا كان هناك كود مكرر أو دوال مستنسخة بدون تنسيق صحيح

⚠️ 5. فحص احتمالية وجود خلل أو تضارب في الترابط:
التركيز على:

هل هناك دوال مستنسخة أكثر من مرة تسبب تعارض؟

هل هناك مشكلة في ترتيب تحميل الأكواد (مثلاً، JavaScript يتم تحميله قبل HTML المطلوب)؟

هل جميع الأحداث مربوطة بشكل صحيح؟

هل هناك مشكلة في التواصل بين أقسام الصفحة (مثل السياسات - الاجتماعات - الجوازكريت) تؤدي إلى تعطل وظائف أخرى؟

هل الأكواد المعدّلة سابقًا تم ربطها فعليًا أم فقط تم تعديل ملفات دون ارتباط حقيقي بالصفحة؟

📝 6. استخراج تقرير شامل:
لا تُجري أي تعديل الآن

قدم تقريرًا واضحًا يحتوي على:

البنية العامة للصفحة

طريقة التفاعل بين الملفات

كيفية انتقال البيانات

مصدر أي خلل محتمل في الترابط أو استنساخ الدوال

النقاط الحساسة التي يمكن أن تتأثر عند التعديل القادم

🔧 7. ملحوظة خاصة:
تم سابقًا طلب تعديلات تتعلق بنظام الولايات، الأسعار، التوصيل…
ولكن التعديلات لم تعمل كما هو متوقع رغم تأكيد تنفيذها، مما يرجّح احتمال:

وجود خلل في الترابط العام للصفحة

أو مشكلة في استنساخ أو تكرار الدوال بشكل غير منظم

لذلك، هذه المهمة ضرورية لفهم أصل المشكلة قبل أي تعديل جديد.
